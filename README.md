## Running Locally

```bash
# Clone the repo
git clone https://github.com/Nusaibah-Rawnak/nusaibah-rawnak.github.io.git
cd nusaibah-rawnak.github.io

# Install dependencies
npm install

# Start dev server
npm run dev
```

The site will be available at `http://localhost:4321`.
